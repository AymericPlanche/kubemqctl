## kubemqctl cluster events

Show Kubemq cluster events command

### Synopsis

Events command allows to show a real-time Kubemq cluster events

```
kubemqctl cluster events [flags]
```

### Examples

```

 	# Show Kubemq cluster events
	kubemqctl cluster events

```

### Options

```
  -h, --help   help for events
```
